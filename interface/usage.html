How to search the corpus



  • 1. Select a named entity from a drop-down list.

    You can choose from the following entities: Court Name, Appellant Name, Respondent Name, Judge Name, Counsel Appellant, Counsel Respondent, Citation, Section, Date of hearing, Date of judgment (e.g., Judge Name).


    2. Enter a keyword and click the Submit button.

    You can search for one entity or multiple entities in the corpus.

    • Search by one entity: enter a keyword (e.g., Harris) and click the Submit button.
    • Search by multiple entities: enter a keyword (e.g., Harris) and click the Add button. You will see one more line with a drop-down list for entities. Again, select a named entity, enter a keyword for that entity, and click the Submit button.
    • If you want to delete any entity and keyword from multiple entities you have chosen - click the Remove button.

    3. Examples of keywords by entity types:
    • Court Name: "The Supreme Court of British Columbia", "THE COUNTY COURT OF VANCOUVER", "the Court of Appeal", etc.
    • Appellant Name: "Lloyd", "Mr. Lloyd", "Fransen Insurance Services Ltd.", etc.
    • Respondent Name: "Regina", "InsureBC Underwriting Services Inc.", etc.
    • Judge Name: "The Honourable Madam Justice D. Smith", "Smith", "SMITH J.A", etc.
    • Counsel Appellant: "Fai", "D.N. Fai", etc.
    • Counsel Respondent: "Walker", "J.N. Walker", etc.
    • Citation: "R. v. Lloyd", "R. v. Thompson, 2019 BCCA 1", etc.
    • Section: "Section 253", "Sec 253", "S. 255"., etc.
    • Date of hearing: "January 2, 2019", "Dec", "14", etc.
    • Date of judgment: "January 20, 2019", "2019", "Jan", etc.

    4. Returned documents.

    After clicking the Submit button, you will notice a table with returned documents.

    • The column Matched Document returns unique instances of the occurrence of a particular keyword for a given entity for every document.
    • The column Judgement link takes you to the actual Court Judgement document of the match on the BC Law webpage.
